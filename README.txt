# ScaryGame
 MoonMoon GameJam
https://github.com/Kaine-R/ScaryGame

To Start Game:
Unzip the Scary Game Folder
Find Scary Game application (.exe)
If you want to move game somewhere else, bring the sounds, and images folder. It sould still work

Controls:
"right click\" to shoot weapon"
"wasd\" to move"
"Tab\" to switch weapons:"
"Zero (0)\" for fast quit
"e\" to pick up close weapons and medpacks
"escape\" to pause"
"There's no settings, or volume control and it's barely a game"
